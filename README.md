# zhihu-story-agent 知乎故事智能体

## 📌 项目简介
本项目是一款基于 **RAG 架构 + Agent 智能体** 的知乎故事创作辅助工具，专为知乎平台打造。它能根据用户提供的主题、人设和风格设定，自动完成故事大纲、情节填充、文案润色全流程，输出符合知乎调性的爆款故事内容，帮创作者高效实现内容生产。

## ✨ 核心功能
- **人设化创作**：内置「反恋爱脑、有人文情怀」的知乎爆款人设，生成内容自带观点和共鸣感
- **多风格改写**：支持将草稿一键转化为知乎体、现代诗、小红书笔记等多种文风
- **RAG知识库增强**：可接入外部故事素材，实现精准情节引用、内容溯源
- **全流程创作**：大纲生成、剧情扩写、反转结尾、话题标签推荐一站式搞定

## 🛠️ 技术栈
- Python 3.8+
- `requests`：API接口网络请求
- `openai`：大语言模型智能调用
